# 🎮 Hidden Number Game (Python Project)

A fun multiplayer command-line number guessing game built using Python.

Players try to guess a hidden random number between **1 and 100**.  
The game gives hints like:

- 📈 Higher
- 📉 Lower

until a player guesses the correct number.

---

# 📌 Features

✅ Multiplayer Support (1–50 Players)  
✅ Random Number Generation  
✅ Hint System (Higher / Lower)

# 🎲 Game Rules

- The system generates a random number between **1 and 100**
- Players take turns guessing the number
- The game provides hints:
  - `Higher` → Guess bigger number
  - `Lower` → Guess smaller number
- First player to guess correctly wins 🎉

---

# 👥 Multiplayer System

- Minimum Players: 1
- Maximum Players: 50

Each player enters:
- Player Name
- Guess Number
